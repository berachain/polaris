# Polaris Ethereum JSON-RPC

The Cosmos integration of the Polaris Ethereum adds additional JSON-RPC namespaces for Cosmos related functionality.

## Namespaces

### `cosmos` Namespace
The `cosmos` API gives access to several non-standard RPC methods to inspect the contents of
several [Cosmos SDK Modules](https://docs.cosmos.network/main/building-modules/intro). The `cosmos` namespace definitions are coming soon.

### `comet` Namespace
The `comet` API gives access to several non-standard RPC methods to inspect the contents of
the [CometBft Consensus Engine](https://github.com/cometbft/cometbft). The `comet` namespace definitions are coming soon.
